## Getting started

<Steps>
  <Step title="Edit this page">
    Click anywhere and start typing. Changes save and publish automatically.
  </Step>
  <Step title="Create a new page">
    Click **+ Add page** in the sidebar navigation.
  </Step>
  <Step title="Make it yours">
    Go to **Settings** to set your logo, colors, and custom domain.
  </Step>
  <Step title="Invite your team">
    Add team members from **Members** in the sidebar. They can edit too.
  </Step>
</Steps>

## Editor tips

- Type `/` on an empty line to see all available blocks
- Select text and press `Cmd+K` to add a link
- Press `Cmd+S` to save manually (auto-save is on by default)
- Switch between **Visual** and **Source** mode in the toolbar
